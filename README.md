# 📦 JavaScript Projects

Coleção de projetos práticos desenvolvidos com JavaScript, com foco em evolução de lógica, manipulação do DOM e construção de aplicações reais.

* Evoluir organização e estrutura de código

## 📁 Projetos

### 📝 Lista de Tarefas (To-Do List)

https://github.com/Natan-devDesign/JavaScript_Projects/tree/main/01_listaDeTarefas

* Remover tarefas

* CSS
